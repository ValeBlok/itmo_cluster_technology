# Развертывание кластера

Разворачивание происходит с помощью kubeadm.

## Инфраструктура

Перед запуском нужно 4 виртуальные машины: 1 control plane + 3 worker nodes. Минимальные требования: 2 vCPU, 2 ГБ RAM, находятся в одной сети.

Локально должен быть установлен ansible.

## Настройка конфигурации проекта

### `inventory/hosts.ini` (обязательная конфигурация)

Вписать IP адреса машин + изменить переменные.

### `group_vars/all.yml`

Все переменные в этом файле не обязательны к настройке.

## Запуск

Установка зависимостей:

```bash
ansible-galaxy collection install -r requirements.yml
```

Проверка связи с нодами:

```bash
ansible all -m ping
```

Запуск плейбука:

```bash
ansible-playbook site.yml
```

Если что-то пошло не так - откат плейбука:

```bash
ansible-playbook reset.yml
```
